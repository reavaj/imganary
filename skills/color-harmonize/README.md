# Color Harmonize Skill

Recolors images using harmonious color palettes via GIMP Python-Fu batch automation.

- Analyzes dominant hue via pixel sampling and color quantization
- Constructs palettes using color theory: complementary, analogous, split-complementary, tetradic
- Remaps hue channel while preserving luminance and saturation
- Runs GIMP in headless batch mode — no GUI needed
