## How to Add Entries

When a new skill generates a script, add a row to the table above:

1. **Skill** — Name of the Claude Code skill
2. **Category** — Skill category (`gimp-core`, `image-analysis`, `workflow`)
3. **Script(s) Generated** — Path(s) under `scripts/` that the skill produces
4. **Description** — What the skill/script does
