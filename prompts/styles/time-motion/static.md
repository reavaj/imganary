# Static

Perfectly locked-off, unmoving, surveillance-like stillness — Ozu's tatami shots, security cameras, Michael Haneke.

## Visual DNA
- **What it does:** Fixes the viewpoint with absolute rigidity, creating an unblinking, observational, almost voyeuristic quality where the frame never moves and the subject must exist within it.
- **Key characteristics:**
  - Camera absolutely locked, no movement
  - Compositional rigidity, geometric precision
  - Observational, detached, clinical
  - Unsettling patience — the camera simply watches
  - Subjects enter and exit the fixed frame
- **Reference points:** Yasujirō Ozu (tatami-level shots), Michael Haneke (Funny Games, Caché), security camera footage, tableau vivant photography, Roy Andersson
- **Pairs well with:** symmetrical, austere, eerie, fluorescent

## FLUX keywords
static, still, unmoving, locked camera, surveillance, fixed frame, tableau, motionless, frozen scene, security camera, perfectly still
