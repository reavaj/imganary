# Layered

Multiple depth planes creating visual richness — Welles' deep staging, collage art, theatrical dioramas.

## Visual DNA
- **What it does:** Activates foreground, midground, and background simultaneously, creating visual depth and density through stacked planes of interest.
- **Key characteristics:**
  - Distinct foreground, midground, and background elements
  - Each layer contains meaningful visual information
  - Parallax depth and spatial richness
  - Eye travels through the image in depth
  - Dense but organized visual field
- **Reference points:** Orson Welles (Citizen Kane deep staging), collage art (Hannah Höch), theater set design, diorama photography, Jean-Pierre Jeunet
- **Pairs well with:** deep-focus, intricate, cluttered, panoramic

## FLUX keywords
layered composition, multiple layers, depth planes, foreground background, deep staging, layered depth, parallax, stacked planes
