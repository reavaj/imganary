# Monochromatic

A single hue in varying tones — cyanotype blue, sepia warmth, Picasso's Blue Period, infrared red.

## Visual DNA
- **What it does:** Restricts the entire image to one color family, using only variations in lightness and saturation to create form, unifying the image under a single chromatic identity.
- **Key characteristics:**
  - One dominant hue across the entire frame
  - Tonal range from dark to light within that hue
  - Strong visual unity and cohesion
  - Emotional coding through color choice (blue = melancholy, red = intensity)
  - Simplified palette focuses attention on form and light
- **Reference points:** Cyanotype prints, sepia photography, Picasso Blue Period, Yves Klein blue, infrared photography
- **Pairs well with:** minimal, serene, austere, graphic
