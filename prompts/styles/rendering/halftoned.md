# Halftoned

Image built from regular dot screens at varying sizes — newsprint, Lichtenstein, Ben Day dots, screen printing.

## Visual DNA
- **What it does:** Converts continuous tone into a grid of varying-size dots, revealing the mechanical reproduction process and creating a retro, mass-produced, pop-art texture.
- **Key characteristics:**
  - Regular dot grid visible at close range
  - Dot size varies to create tonal range
  - CMYK process color visible as overlapping screens
  - Mechanical, mass-produced, democratic
  - Retro print aesthetic, pre-digital
- **Reference points:** Roy Lichtenstein (pop art), newsprint photography, Ben Day dots, screen printing, 1960s comic books
- **Pairs well with:** graphic, stippled, nostalgic, saturated

## FLUX keywords
halftone, halftoned, dot screen, Ben Day dots, newsprint, Lichtenstein dots, print screen, CMYK dots, retro print, pop art dots
