# Imganary — AI-Driven GIMP Automation Framework

A structured, extensible automation framework integrating Claude Code with GIMP to enable AI-driven image editing workflows. The system generates reusable scripts, grows a persistent library, and is version-controlled for long-term storage and community sharing.

## Core Goals

- Automate GIMP image manipulation using Claude Code skills
- Generate and save reusable GIMP Python-Fu/Script-Fu scripts to a growing library
- Integrate local image recognition models so Claude understands image context before editing
- Maintain everything in a well-organized repo for long-term storage and potential community sharing

## Repository Structure

```
imganary/
├── skills/               # Claude Code skill definitions
│   ├── gimp-core/        # CLI and Python-Fu wrappers for common ops
│   ├── image-analysis/   # Local vision model invocation
│   └── workflow/         # Pipeline orchestration and batch processing
├── scripts/              # Generated GIMP Python-Fu scripts
│   ├── color/
│   ├── transform/
│   ├── filters/
│   └── compositing/
├── models/               # Local vision/recognition models + setup guide
├── docs/
│   ├── skill-index.md    # Maps skills to the scripts they generate
│   └── setup.md          # Environment setup instructions
└── README.md
```

## How It Works

1. **Analyze** — A local vision model (LLaVA, YOLO, CLIP) examines the image and returns a scene description
2. **Generate** — Claude uses the scene context to generate a targeted GIMP Python-Fu script
3. **Execute** — The script runs in GIMP to perform the transformation
4. **Save** — The script is saved to the library under the appropriate category

## Environment Requirements

- Claude Code (CLI)
- GIMP with Python-Fu support enabled
- Python 3.x
- Ollama (for running LLaVA locally)
- Git

See [docs/setup.md](docs/setup.md) for detailed setup instructions.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/<your-username>/imganary.git
cd imganary

# Install Ollama and pull LLaVA
brew install ollama
ollama pull llava

# Verify GIMP Python-Fu is available
gimp -i -b '(gimp-version)' -b '(gimp-quit 0)'
```
